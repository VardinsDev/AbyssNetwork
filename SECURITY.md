# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue**.

Instead, report it privately by emailing or sending a Discord DM to **VardinsDev**.

You can expect a response within 48 hours. If the vulnerability is confirmed it will be patched in the next release and you will be credited unless you prefer to remain anonymous.
